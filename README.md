# Sandbox up and runnning with Serverless Functions (Jamstack Explorers)

# Setup

- Install node.js

- Install netlify-cli as a global depedency
```bash
npm i netlify-cli -g
```

- Install project depedency
```bash
npm i
# or
npm install
```

- add `.env` to root app directory


- run app
```bash
ntl dev
```
